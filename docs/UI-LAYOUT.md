## 1. โครงหน้า (App Shell)
```
┌────────────┬──────────────────────────────────────────────────┐
│            │  Hello, {ชื่อ} 👋        [ค้นหา]    💬 🔔  {โปรไฟล์▾} │ ← Topbar
│  SIDEBAR   ├──────────────────────────────────────────────────┤
│            │  [KPI] [KPI] [KPI] [KPI]                          │ ← Row KPI (4 การ์ด)
│  (โลโก้)   ├───────────────────────────┬──────────────────────┤
│            │  กราฟเส้น    │ โดนัท       │  ปฏิทิน + ตารางนัด    │ ← Row กราฟ + พาเนลขวา
│  เมนูกลุ่ม  ├───────────────────────────┤  (Schedule/Upcoming) │
│            │  ตารางจัดการ (list+filter) │                      │ ← Row ตาราง
│            │                           │                      │

└────────────┴──────────────────────────────────────────────────┘
```

หลักการ: การ์ดมุมโค้ง (rounded-xl), เงานุ่ม (soft shadow), เว้นช่องเยอะ, พื้น `#f7f8fa`, การ์ดพื้นขาว

## 2. Sidebar (ซ้าย)
- **โลโก้ "BENJAMIN"** ด้านบน (โทน Dark Blue)
- เมนูแบ่งกลุ่มมีหัวข้อย่อย (เหมือน General/Reports/Settings ใน ref)
- ไอคอน + ข้อความ, **active = พื้น Dark Blue + ตัวอักษรขาว** (pill โค้ง), hover = Dark Blue 50
- มี badge ตัวเลขข้างเมนู (เช่น ลีดใหม่ 6) เหมือน "Appointment ⑥" ใน ref
- **ปุ่ม "ออกจากระบบ"** + ลิงก์ "เยี่ยมชมเว็บไซต์" ด้านล่างสุด

### เมนูตาม role
| กลุ่ม | ดีลเลอร์ | HQ (เพิ่ม) |
|------|----------|-----------|
| **งานหลัก** | แดชบอร์ด, ลีด ⑥, ลูกค้า, ใบเสนอราคา, โครงการ | + Lead Pool, อนุมัติ ⑥ |
| **รายงาน** | Analytics, การเงิน | (รวมทั้งเครือ) |
| **จัดการ** | ตั้งค่า, ผู้ใช้ทีม | + จัดการดีลเลอร์, Master/ราคา |
| **ช่วยเหลือ** | Help & Support, ตั้งค่า | |

## 3. Topbar
- ซ้าย: **"สวัสดี, {ชื่อ} 👋"** + บรรทัดรอง ("ยินดีต้อนรับสู่ระบบจัดการ Benjamin")
- กลาง: ช่องค้นหา ("ค้นหาลีด/ลูกค้า/โครงการ…")
- ขวา: 💬 ข้อความ · 🔔 แจ้งเตือน (badge) · ชิปโปรไฟล์ (รูป + ชื่อ + role + ▾)

## 4. Row KPI (4 การ์ด) — แบบเดียวกับ ref
แต่ละการ์ด: ไอคอนในกล่องโค้ง + ป้ายชื่อ + **ตัวเลขใหญ่** + badge % เทียบเดือนก่อน + "เดือนที่แล้ว"

| | ดีลเลอร์ | HQ |
|---|---------|-----|
| 1 | 🎯 เป้า vs ยอดขาย | 💰 ยอดขายรวม |
| 2 | 💰 มูลค่า Pipeline | 📊 Pipeline รวม |
| 3 | 📈 Win Rate | 🏗️ โครงการ Active |
| 4 | 🏗️ โครงการ Active | ⏱️ On-time % |

> สี badge %: เพิ่ม = success เขียว, ลด = danger แดง

## 5. Row กราฟ + พาเนลขวา

### 5.1 กราฟเส้น (ซ้าย) — แทน "Patients Statistics"
- **ยอดขาย/ลีดรายเดือน** (Jan–Dec) + dropdown ช่วงเวลา (รายเดือน/ไตรมาส)
- เส้น/พื้นไล่เฉด **Dark Blue**, จุด tooltip ตัวเลข
- ใช้ Recharts/Tremor

### 5.2 โดนัท (กลาง) — แทน "Total Expense"
- **สัดส่วน Pipeline ตามสถานะ** (หรือ รายได้/ต้นทุน/อื่นๆ) + ตัวเลขกลางวง (%)
- โทน Dark Blue / Steel / Silver ไล่ระดับ + legend จุดสี

### 5.3 พาเนลขวา — แทน "Calendar + Schedule + Upcoming"
- **แถบสัปดาห์** (จ.–ศ. + วันที่) เลือกวัน + ปุ่ม "+ เพิ่มนัด"
- **Schedule วันนี้**: นัดสำรวจไซต์/นัดลูกค้า (จาก `Activity.scheduledAt`) — เวลา + ชื่อ + สถานที่
- **Upcoming**: นัด/ส่งมอบที่จะถึง (Milestone/Handover) — ดึงจาก Project/Activity
- "ดูทั้งหมด" ลิงก์มุมขวาแต่ละบล็อก

## 6. Row ตาราง — แทน "Patients Management"
- หัวข้อ + ลิงก์ "ดูทั้งหมด"
- แถวเครื่องมือ: ช่องค้นหา + ปุ่ม **"ตัวกรองขั้นสูง"** (พื้น Dark Blue)
- ตาราง: checkbox + คอลัมน์ **เรียงได้** (↕) + **badge สถานะ** + ลิงก์ "รายละเอียด"

### ใช้กับ
| หน้า | คอลัมน์ |
|------|---------|
| **ลีด** | ID · ชื่อลูกค้า · จังหวัด · สินค้า · สถานะ(badge) · มูลค่า · รายละเอียด |
| **โครงการ** | เลขที่ · ชื่อ · ลูกค้า · เฟส(1-6) · สถานะ · ส่งมอบ · รายละเอียด |
| **ใบเสนอราคา** | เลขที่ · ลูกค้า · ยอด · สถานะอนุมัติ · วันหมดอายุ · รายละเอียด |

> badge สถานะใช้สีฟังก์ชัน (เขียว/เหลือง/แดง) + Dark Blue สำหรับสถานะทั่วไป

## 7. คอมโพเนนต์ที่ต้องสร้าง (Component Inventory)
| Component | ใช้ที่ |
|-----------|--------|
| `<AppSidebar>` (กลุ่มเมนู + active + badge) | ทุกหน้า |
| `<Topbar>` (greeting + search + noti + profile) | ทุกหน้า |
| `<KpiCard>` (icon, label, value, delta%) | แดชบอร์ด |
| `<LineChartCard>` / `<DonutCard>` | แดชบอร์ด |
| `<SchedulePanel>` (week strip + list) | แดชบอร์ด |
| `<DataTable>` (sort, checkbox, filter, status badge, pagination) | ลีด/โครงการ/ใบเสนอราคา |
| `<StatusBadge>` | ทั่วระบบ |
| `<PageShell>` (layout 2 คอลัมน์) | ทุกหน้าใน (app) |

> ส่วนใหญ่ map กับ shadcn/ui: `card, button, input, table, badge, dropdown-menu, avatar, tabs, calendar`
